## 🖥️ How to Use

### For Users:
1. Double-click **START.bat**
2. Login with `admin` / `12345`
3. Start calculating bills!

### For Developers:
1. Modify code in `src/`
2. Run START.bat (auto-compiles)
3. Test your changes
